## t
```re sig
type t('a) =
  | Undefined: t(undefined_val)
  | Null: t(null_val)
  | Boolean: t(bool)
  | Number: t(float)
  | String: t(string)
  | Function: t(function_val)
  | Object: t(obj_val)
  | Symbol: t(symbol);
```
