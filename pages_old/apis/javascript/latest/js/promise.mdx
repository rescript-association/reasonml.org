# Promise

<Intro>

Provide bindings to JS promise.

</Intro>

## t

```re sig
type t(+'a);
```

## error

```re sig
type error;
```

## make

```re sig
let make: ((~resolve: (. 'a) => unit, ~reject: (. exn) => unit) => unit) => t('a);
```

## resolve

```re sig
let resolve: 'a => t('a);
```

```re sig
let reject: exn => t('a);
```
